StudyMatch

Topic

StudyMatch is a website where college students can find study partners or groups that are taking the same classes as them.

Audience

My Audience is current college students.

Database

The database will store user profiles (name, year, courses, university), courses, study groups, and scheduled study sessions. It will also track user participation in study groups, messages exchanged within groups, and session details like date, time, and location (virtual or in-person). The data will be dynamically added to the database as users interact with the platform (profiles, group creation, session scheduling).

Design & Style

The design of StudyMatch follows a modern and inviting aesthetic, using a vibrant blue (StudyMatch title color) as the primary color to represent focus and collaboration. It is complemented by a warm coral (subheading color) for accents, a dark pink for buttons (button above) a soft gray (paragraph color) for text, and a light beige (background color) background to create a clean and user-friendly interface.

Scope

StudyMatch will have 5 pages.
  1. Home Page – Overview of the website, introduction, and button to sign up/login.
  2. Create/Join Study Session Page – Allows users to create new study sessions or join existing ones. Also has a chat with study group feature.
  3. Profile Page – Users can manage their courses, study preferences, and personal info.
  4. Study Groups Page – Displays a list of available study groups with filtering/search options.
  5. Login/Signup Page – To make sure students only can access platform (.edu) email and to save their profile, chat history, and study groups efficiently.
